# 飛行シミュレーションゲーム 要件定義

## 概要

翼の形状・位置、重心、機体寸法などのパラメータをGUIで直感的に設計し、飛行力学に基づいたリアルタイム飛行シミュレーションを体験できるデスクトップアプリケーション。
空力モデルは文献値を参照した精度の高い実装を目指し、知識の蓄積に伴い段階的にアップデートする。

## 2. 空力モデル

### 2.1 Phase 1：平板モデル（薄翼理論）

平板の揚力傾斜（薄翼理論）：

```text
CL_alpha = 2π  [/rad]  （理論値）
CL = CL_alpha × α  （失速前の線形領域）
```

有限翼補正（Prandtl 揚力線理論）：

```text
CL_alpha_finite = (2π × AR) / (2 + sqrt(4 + AR²))  （Helmbold の式）
```

抗力（誘導抗力 + 有害抗力）：

```text
CD = CD0 + CL² / (π × AR × e)
```

- `CD0`：有害抗力係数（平板の場合は摩擦抗力主体、初期値 0.01〜0.02）
- `e`：Oswald 効率（初期値 0.85）

### 2.2 Phase 2：NACA翼型データ参照

- NACA 0012 等の CL-α、CD-α テーブルデータをファイルから読み込み
- α の範囲：-20° 〜 +20°（失速域を含む）
- 線形補間でリアルタイム計算
- 失速特性（非線形領域）の再現

### 2.3 ピッチングモーメント

```text
Cm = Cm0 + Cm_alpha × α + Cm_delta_e × δe
```

重心まわりのモーメント（主翼 + 水平尾翼の寄与）：

```text
Cm_cg = CL_w × (x_ac_w - x_cg)/c̄  +  CL_t × (x_ac_t - x_cg)/c̄ × (S_t/S)
```

- `x_ac_w`：主翼の空力中心（通常 25%MAC 付近）
- `x_ac_t`：尾翼の空力中心

### 2.4 大気モデル（国際標準大気 ISA）

```text
T(h) = 288.15 - 0.0065 × h  [K]  （対流圏、h < 11000m）
P(h) = 101325 × (T/288.15)^5.2561  [Pa]
ρ(h) = P / (R × T)  [kg/m³]  (R = 287.05 J/kg·K)
```

## 3. 安定性解析

### 3.1 静安定

#### 縦の静安定（ピッチ安定）

```text
静安定マージン SM = (x_ac - x_cg) / c̄ × 100  [%MAC]
```

- SM > 0：静安定（x_cg が x_ac より前）
- SM = 0：中立安定
- SM < 0：静不安定

中立点（Neutral Point）の計算：

```text
x_np = (CL_alpha_w × x_ac_w + CL_alpha_t × x_ac_t × (S_t/S) × η_t)
     / (CL_alpha_w + CL_alpha_t × (S_t/S) × η_t)
```

- `η_t`：尾翼動圧比（ダウンウォッシュ考慮、初期値 0.9）

横・方向安定 → 3D Phase で追加

### 3.2 動安定（2D：縦の動安定）

線形化した縦の運動方程式（小擾乱理論）：

状態変数：`x = [u, w, q, θ]`（速度擾乱・迎え角擾乱・ピッチ角速度・ピッチ角）

システム行列 A を構成し固有値解析：

```text
λ = eigenvalues(A)
```

#### 縦の固有モード

| モード | 特徴 | 安定条件 |
| --- | --- | --- |
| フゴイド (Phugoid) | 速度・高度が緩やかに振動（低周波） | Re(λ) < 0 |
| 短周期 (Short Period) | ピッチ・迎え角の速い振動（高周波） | Re(λ) < 0 |

安定微係数（主なもの）：

- `Zu, Zw, Zq, Zde`：Z 方向力の偏微分
- `Mu, Mw, Mq, Mde`：ピッチングモーメントの偏微分

#### 3D Phase 追加モード

- ダッチロール (Dutch Roll)
- スパイラル (Spiral)
- ロール収束 (Roll Subsidence)

## 4. 運動方程式（2D 縦断面）

### 4.1 2D（状態変数：x, z, u, w, q, θ）

並進（ボディ座標系）：

```text
m(u̇ + qw) = T cos(α_T) - D cos(α) - L sin(α) - mg sin(θ)
m(ẇ - qu) = T sin(α_T) + L cos(α) - D sin(α) - mg cos(θ)
```

回転：

```text
I_yy × q̇ = M  （重心まわりのピッチングモーメント）
```

姿勢更新：

```text
θ̇ = q
```

地上固定座標系への変換：

```text
ẋ_e =  u cos(θ) + w sin(θ)
ż_e = -u sin(θ) + w cos(θ)
```

数値積分：RK4（4次Runge-Kutta）を使用（Δt = 1〜10 ms）

## 5. GUI設計（機体デザインUI）

### 5.1 概要

- Dear ImGui（C++）ベースの即時モードGUI
- 2ペイン構成：**設計パネル（左）** + **シミュレーションビュー（右）**

### 5.2 設計パネル

- 翼・尾翼・胴体の各パラメータをスライダー / 数値入力で調整
- パラメータ変更はリアルタイムで機体形状と安定性評価に反映

### 5.3 機体形状ビュー（2Dドローイング）

- 側面図（Side View）：機体の縦断面形状
  - 翼・尾翼の位置と大きさを矩形で表示
  - 重心（CG）・空力中心（AC）・中立点（NP）をアイコンで表示
- 上面図（Top View）：翼幅・後退角・上反角の確認用（Phase 2）
- ドラッグ操作で翼・尾翼位置を直接変更可能

### 5.4 安定性インジケーター

- 静安定マージン（SM）の数値とゲージ表示（緑：安定 / 赤：不安定）
- 失速速度 (V_stall) の表示
- フゴイド・短周期の固有値（ダンピング比・固有振動数）の表示

### 5.5 シミュレーション操作

- **Play/Pause/Reset** ボタン
- 操縦入力：
  - キーボード：昇降舵（↑↓ / W/S）、スロットル（+/-）
  - スライダー：昇降舵偏角・スロットル開度のリアルタイム表示

### 5.6 飛行ビュー（シミュレーション画面）

- 2D サイドビュー（機体・地面・高度グリッド）
- HUD オーバーレイ：対気速度・高度・ピッチ角・昇降率・迎え角・G荷重
